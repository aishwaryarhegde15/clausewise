# ClauseWise – AI Legal Document Analyzer

Upload a PDF/DOCX/TXT → auto split into clauses → classify (e.g., Indemnity, Termination) →
plain-English rewrite → risk score → obligations → export JSON/CSV.

- **Model**: `ibm-granite/granite-3.2-2b-instruct`
- **Views**: Table + Card
- **Exports**: JSON + CSV
- **Optional**: Set `ADAPTER_REPO` in Space Settings to load a LoRA adapter.

> Educational aid, not legal advice.
